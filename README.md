# CLiply

ClipLy is a Chrome extension and intercative web app designed to enable users to save, organize, and manage bookmarks efficiently.
Users can save any webpage with a single click, capturing the URL, title, and a brief description. Users can edit bookmark details post-creation to accommodate any changes.

## Workflow
